# CSS Flexbox



### 1. 특징

- 복잡한 계산 없이 간단함
- 매우 유연하며 반응형 구현 편리
- 정렬이 매우 간단
- 기존의 중앙정렬 번거로움이 해결


### 2. 배우는 이유

- 깔끔, 심플을 추구하는 요즘 트렌드에 적합
- 웹 어플리케이션에 어울리는 레이아웃
- 라이브러리, 프레임워크 없이 반응형 편리하게 구현
- *React Native 에서 Flexbox 방식의 Style Sheet 사용
